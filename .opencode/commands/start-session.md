This is a course for learning the Laravel AI SDK. The course overview is specified in /ai-dourse/course-content.md and for each day, we have a document like /ai-course/day-1.md, day-2.md, etc.
Everyday, while the user starts the day's lesson, you'll have to review their tasks, response to questions and fix daily course's issues if present, updating the course daily content if necessary. 
Once the user finishes the day's lesson, you'll have to generate the contents for the next day as a .md file in the folder /ai-course.
